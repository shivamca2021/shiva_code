Odoo App for Product Variants Description

Odoo Description for Product Variants Apps helps you to add different description for each and every product variant of product template. By default odoo does not allow to add separate description for each product variant , by default it takes product template description all variants automatically which use on sales, purchase, picking . When user select product variant on sale, purchase and picking assigned description will be automatically comes on sale, purchase and picking line and it will be printed on report as well.
